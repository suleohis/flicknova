**A cinematic movie & TV discovery app** inspired by Netflix + Letterboxd ‚Äî built 100% with Flutter for cross-platform excellence (Android, iOS, web, desktop).

### Why This Project?
As a mid-level Flutter developer (4+ years), this MVP showcases my evolution in state management (from GetX to Riverpod), modern UI/UX patterns, performant API handling, and clean architecture ‚Äî skills I've applied to deliver 99% crash-free production apps with 20%+ user engagement gains.

### Key Features
- **Powerful discovery** ‚Äî Search, filter, and browse trending/popular movies & TV shows via TMDB API
- **Infinite pagination** ‚Äî Smooth endless scrolling with shimmer loading states
- **Cinematic UI** ‚Äî Dark mode with glassmorphism, blur effects, hero transitions, and blue/purple accents
- **Offline watchlist** ‚Äî Add/remove titles, view stats, persists with Isar (fast local DB)
- **Cross-platform** ‚Äî Runs beautifully on mobile, web, macOS, Windows
- **Clean & scalable** ‚Äî Feature-first architecture, Riverpod for reactive state, proper error handling & loading states

### Tech Stack
- **Flutter** (latest stable) + **Dart**
- **State Management**: Riverpod (modern, testable, performant)
- **API**: TMDB (The Movie Database) with Dio or http client
- **Local Database**: Isar (lightning-fast, NoSQL for offline data)
- **UI Polish**: Custom themes, animations (Hero, AnimatedSwitcher), shimmer, glassmorphic effects
- **Other**: intl for formatting, flutter_launcher_icons, etc.

### Getting Started (for reviewers/recruiters)
1. Clone the repo: `git clone https://github.com/suleohis/flicknova.git`
2. Get your free TMDB API key ‚Üí https://www.themoviedb.org/
3. Create `.env` file in root with: `TMDB_API_KEY=your_key_here`
4. Install dependencies: `flutter pub get`
5. Run: `flutter run`

(Uses `flutter_dotenv` or similar ‚Äî never commit keys!)
